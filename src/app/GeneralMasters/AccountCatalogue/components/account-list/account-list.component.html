Datos adicionales

Para importar el catálogo de cuentas, por favor, utilice un archivo de Excel (.xlsx) con el siguiente formato y consideraciones.

Columnas Requeridas

El archivo debe contener las siguientes columnas en la primera fila:

  • Código
  • Nombre
  • Naturaleza (Débito o Crédito)
  • Estado Financiero (Estado de situacion financiero o Estado de resultados)
  • Clasificación

Consideraciones Importantes

  • Asegúrese de que no haya filas completamente vacías entre los registros.
  • Todos los campos son obligatorios. Las filas con alguna celda vacía no serán importadas.
  • La jerarquía de las cuentas se creará automáticamente a partir de los códigos.
